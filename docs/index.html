scroll-animation

Animate elements based on the scroll position. 2k minified & gzipped.

Why?

I couldn't find an npm package that fit my needs:

  • ability to animate any DOM property
  • using requestAnimationFrame for performace
  • suspending requestAnimationFrame after a while to minimize load
  • no data- attributes or special tags
